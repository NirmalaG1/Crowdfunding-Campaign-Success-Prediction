Project Title: Crowdfunding Campaign Success Prediction

Problem Statement
Crowdfunding platforms host thousands of campaigns across categories such as technology, education, healthcare, and creative arts. However, not all campaigns achieve their funding goals. Businesses and investors want to understand which campaign factors influence success and which machine learning model can best predict campaign outcomes.


In this project, you will build and compare multiple classification models to predict whether a crowdfunding campaign will be Successful or Unsuccessful based on campaign-related features such as funding goal, campaign duration, backer engagement, and pledged amount.
The project focuses on model evaluation, cross-validation, hyperparameter tuning, and selecting the best-performing classification model.


Dataset Link: Crowdfunding Campaign Dataset


Assignment Tasks

Task 1 – Data Exploration & Preprocessing

Load and inspect the dataset
Handle missing values and duplicates
Encode categorical variables
Perform feature selection
Split the dataset into training and testing sets
Apply feature scaling where required


Task 2 – Classification Model Building

Build and train the following classification models:
Random Forest Classifier
KNN
Decision Tree
Naive Bayes Classifier
Support Vector Machine (SVM)
Experiment with different model parameters to improve performance.


Task 3 – Model Evaluation

Evaluate all models using:
Accuracy
Precision
Recall
F1-Score
Confusion Matrix
Compare training and testing performance.

Task 4 – Cross-Validation & Hyperparameter Tuning

Apply Cross-Validation to evaluate model stability
Perform Hyperparameter Tuning using GridSearchCV or RandomizedSearchCV
Identify the best parameters for each model
Task 5 – Model Comparison & Selection
Compare all classification models using evaluation metrics
Identify the best-performing model
Discuss strengths, weaknesses, and possible improvements

Deliverables
Google Colab / Jupyter Notebook with complete code and outputs
Trained classification models
Evaluation metrics and comparison results
Final conclusion identifying the best model
